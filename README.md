## Bugs:
Camera,flashlight,Dex+HDMI Dex and  other minor bugs.
## Other:
You may notice lag when installing it. It will dissapear shortly after.
Phone may overheat when multitasking,gaming.
## How to install:
First, if you do have an SD card, do this steps:
1. Format it to exFAT (needed!)
if you do have the phone do this and add the steps to the External SD Card, if you own one:
1. Enter Download mode by pressing VOL + Bixby + PWR
2. Flash the TWRP file with Odin
3. Copy the Repartitioner file to the Phone/SD.
(If you do have an SD Card, select image, and then SD Card.)
4. Press install and Flash the Repartitioner File.
5. Wait for 2 minutes, you will be redirected to the TWRP.
6. Copy the Cleaner File to the Phone/SD.
7. Flash it. You will be redirected to the Recovery shortly.
8. Copy the ROM File to the Phone/SD.
9. Flash it. It can take a while so be patient.
10. Done! You are Ready to go!
### !!Do not forget that this steps are ONLY for Exynos 8895 Devices!!
